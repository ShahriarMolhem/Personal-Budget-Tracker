# Project: Personal Budget Tracker (Excel 2026)

## Overview
This project is an entry-level Microsoft Excel workbook designed to help individuals effectively manage their monthly finances. By providing a structured method for logging income and expenses, the tool utilizes basic formulas to offer clear insights into spending patterns and overall financial health.

---

## Key Features

*   **Income & Expense Logging:** A user-friendly sheet for recording all financial transactions.
*   **Automated Summary Dashboard:** Instant calculation of total income, total expenses, and net savings.
*   **Spending Breakdown:** Clear visualization of where money is being spent across different categories (e.g., Housing, Food, Entertainment).
*   **"Planned vs. Actual" Comparison:** Immediate feedback on budget adherence.
*   **Visual Alerts:** Conditional formatting to highlight overspending or low savings.

---

## Technologies & Skills Used

*   **Microsoft Excel (2016+ recommended)**
*   **Basic Formulas:** `SUM()`, `AVERAGE()`
*   **Data Management:** Data Validation (for category dropdowns)
*   **Visualization:** Conditional Formatting, basic Excel Charts

---

## How to Use

1.  **Download:** Open the `Personal_Budget_Tracker_2026.xlsx` file in Microsoft Excel.
2.  **Configuration:** Navigate to the `Settings` sheet (if applicable) to customize your spending categories.
3.  **Data Entry:** Move to the `Transactions` tab. Enter each financial event with the date, description, category, and amount.
4.  **Monitor:** Review the `Dashboard` sheet to see an automatically updated summary of your financial status, including your monthly average spend and remaining budget.

---

## Project Goals

*   **Objective:** To empower users with a simple, hands-on tool for personal financial management.
*   **Success Metric:** Provide 100% accurate financial summaries that help users identify and adjust their spending habits.
